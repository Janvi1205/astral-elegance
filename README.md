# Pradeep Malhotra Ji — Trusted Vedic Astrologer

A premium, high-performance web application for Pradeep Malhotra Ji's astrology consultation services. Built with React, TypeScript, GSAP for smooth animations, and optimized for maximum visual impact and speed.

## Key Features
- **Premium Design**: Dark-mode editorial aesthetic with gold-gradient accents.
- **Service Showcases**: Multi-modal service cards with custom AI-generated imagery.
- **Smart Booking**: Integrated WhatsApp consultation inquiry system.
- **Performance Optimized**: Code-splitting, lazy-loading, and LCP image optimizations.
- **SEO Ready**: Semantic HTML and refined metadata for search engine visibility.

## Setup
1. `npm install`
2. `npm run dev`
